# CircLeo - Draw The Perfect Circle
CircLeo is an interactive web game where users try to draw the most accurate circle around a red dot. Test your precision, beat your own high score, and challenge your friends!  

## Features

- Draw circles with your mouse or touch input.
- Live scoring while drawing to track accuracy.
- Best score is stored locally in the browser.
- Clear canvas to start over.
- Change stroke color with smooth transitions.
- Challenge your friends: copy your best score to clipboard with one click, no alerts.
- Responsive layout, works on desktop and mobile.
